# 记录不可思议

> 很难想象我又开新帖了. 这篇会一直更, 记录一些碰到的意想不到的问题.

## float(0_0)?

不难理解, 但很难接受. 在写不规范json解析器的时候碰到的.

~~~ python
float(0_0)  # 0.0
float(0_0_0_0_0_0_0)  # 0.0
~~~

直接让我开了一篇新帖.

~~~ python
float(1_0)  # 10.0
float(12_000) # 12000.0
~~~

这样就好接受多了, 权当自己孤陋寡闻吧.
